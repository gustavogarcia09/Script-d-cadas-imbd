# Script-d-cadas-imbd
IMDb Top 1000 – Métricas por Década
Script em Python para processar o dataset IMDb Top 1000 e gerar métricas agregadas por década de lançamento.
O foco é exclusivamente a limpeza básica dos dados e a separação temporal, sem análise exploratória detalhada.
Funcionalidades
Leitura de arquivo CSV
Limpeza das colunas Released_Year e Runtime
Criação da coluna Decade
Cálculo de métricas por década (quantidade, nota média, duração média)
Exportação dos resultados em CSV e exibição no terminal
